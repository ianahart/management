# CMS NSTAT

# Description
This application was made with Angular and Django. It is an admin dashboard for a college management system. It allows the admin to do a variety of things.

# Features
- Analytics
- Manage Staff
- Manage Courses
- Manage Students
- Manage Classes
- Reports
- Attendance
- Forgot Password
